# 🎬 漫剧分镜脚本生成器

一个用 Python 写的小工具：输入题材和设定，自动生成竖屏漫剧 / 短剧的**分镜脚本**。
每个分镜都带【画面描述】（喂 AI 绘画）和【对白】（喂 AI 配音），可以直接接入漫剧制作流程。

> 我的第一个 Python 项目，学习 + 实战中。

## ✨ 功能

- **模板模式**：从词库随机填空生成骨架，不花钱，用来理解「可替换内容」原理
- **AI 模式**：调用 DeepSeek 真正生成有剧情逻辑的脚本
- **批量量产**：用 `任务清单.txt` 一次性批量生成多条脚本（数据与逻辑分离）
- 结果自动保存到 `output/` 文件夹

## 🚀 怎么运行

1. 安装依赖：
   ```bash
   py -m pip install openai
   ```
2. （想用 AI 模式才需要）配置密钥：把 `config_example.py` 复制成 `config.py`，填入你的 DeepSeek key
3. 单条生成：
   ```bash
   py main.py
   ```
4. 批量量产：编辑 `任务清单.txt` 后运行
   ```bash
   py 批量生成.py
   ```

## 📁 文件说明

| 文件 | 作用 |
|---|---|
| `main.py` | 主程序：单条交互式生成 |
| `批量生成.py` | 读取任务清单，批量量产 |
| `任务清单.txt` | 批量任务样例（题材\|设定\|剧情\|分镜数） |
| `templates.py` | 模板 + 词库（「可替换内容」的核心） |
| `ai_writer.py` | 调用 DeepSeek 的逻辑 |
| `config_example.py` | 密钥配置模板（复制成 config.py 用） |

## ⚠️ 安全提醒
